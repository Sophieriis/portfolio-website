Grundlæggende web

I grundlæggende web skulle vi påbegynde vores rejse med HTML og CSS. Alt fra design af brugergrænseflader til hvordan man benytter sig af Visual Studio Code. Det var dette tema som skabte grundforståelse for hvordan man koder og opbygger en hjemmeside, fra bunden.

Grundlæggende ux

I grundlæggende UX lærte vi at bruge Adobe XD på en helt ny måde, ved at skabe et design som skulle fungere responsivt. Layout og design var elementært, samt brugerens rejse gennem prototypen. Vi testede designs og fandt dermed ud af hvad som skulle ændres og det som fungerede godt.

Animation

I animationstemaet skulle vi lave vores eget animationsspil. Vi blev her introduceret for javaScript og dets måde at puste liv i elementer og animationer. ”Adobe Illustrator” blev også gjort brug af, da vi selv skulle designe vores karakterer og de elementer som skulle indgå.

Grundlæggende indhold

I grundlæggende indhold, skulle vi arbejde sammen som gruppe. Samarbejde, interview, produktion og redigering var hovedpunkterne på dette tema. Vi fik forståelse for videoredigering, interviews og hjemmesideoptimering.
